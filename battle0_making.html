1. Material Prep & Layout

Start by preparing the base material. You will need one sheet of MDF (2400 x 1200 x 6 mm). Quarter this sheet to get four 1200 x 600 mm pieces to work with.

Next, download the design file and open it in Inkscape.

The layout of the components is optimized for a laser cutter with a bed size of 1200 x 600 mm. If your machine's cutting area is different, you will need to rearrange the components to fit.

To prepare the files for cutting, copy the components from each of the four quadrants into a new, separate file. Then, export each new file into DXF format to use with your laser cutter.

2. Assemble the Body

Position the components according to the holes, ensuring they are in the correct sequence. This is a crucial step for the main body assembly.

3. Securing the Body

Prepare 7 Jenga blocks (75 x 25 x 15 mm). Create a Ø6mm, 20mm deep hole in each and insert an M4 insert nut. Secure the body using bolts, nuts, and washers as shown, and then assemble the 7 legs.

4. Stepper Motor sliding module

The frame uses Jenga blocks. To connect the step motor and slider, purchase a 40mm, M3 bolt separately. The previously connected bolt was too short to be used. You can secure the frame with wood glue and additionally drill holes for screws.
